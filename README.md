# Digital Design using Verilog

This repository contains implementations of fundamental digital design components using Verilog HDL.  
It focuses on building a strong foundation in both **combinational** and **sequential** logic, with a structured and modular approach.

---

## 📂 Repository Structure

### 🔹 ALU (4-bit)
Implementation of a 4-bit Arithmetic Logic Unit with multiple operations:

- Logical Operations: AND, OR, XOR, NOR  
- Arithmetic Operations: ADD, SUB  
- Multiplication  
- Comparison  

Includes:
- Modular design (separate units for logic, arithmetic, etc.)
- Flag generation (Carry, Zero, Sign, Overflow)
- Testbench for verification  

---

### 🔹 Registers
Collection of different types of shift registers:

- **PISO (Parallel-In Serial-Out)**
- **SIPO (Serial-In Parallel-Out)**
- **SISO (Serial-In Serial-Out)**
- **Universal Shift Register** (Shift Left, Shift Right, Load, Hold)

Each module includes:
- Verilog source code  
- Testbench for simulation

The goal of this repository is to:

  - Register Files  
  - Datapath Design  
  - Finite State Machines  
  - Simple Processor Design
